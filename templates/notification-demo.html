How to Use

Include the notification CSS and JavaScript files in your HTML:

<link rel="stylesheet" href="/static/styles/notifications.css"> <script src="/static/scripts/notification-system.js"></script>

Then use the notification system in your JavaScript:

// Success notification window.notifications.success({ title: 'Success!', message: 'Your action was completed successfully.' }); // Error notification window.notifications.error({ title: 'Error!', message: 'Something went wrong.' }); // Warning notification window.notifications.warning({ title: 'Warning!', message: 'This action cannot be undone.' }); // Info notification window.notifications.info({ title: 'Information', message: 'Your game will start in 10 minutes.' }); // LARO-themed notification window.notifications.laro({ title: 'LARO Update', message: 'New basketball court added near your location!' });
